# CPK

This program serves as a testbed for interacting with Apple's private APIs.
Specifically, those concerned with emojis and providing the character picker.
This is used to facilitate correct interaction with Tapbacks.

## Frameworks

- EmojiFoundation (`EMF`)
- CoreEmoji (`CEM`)
- CharacterPicker (`CPK`)
  - CharacterPalette (essentially CharacterPicker's "InputMethodKit frontend")

## Defaults Domains

- `com.apple.CharacterPicker`
- `com.apple.EmojiPreferences`

Might be helpful to influence desired interactions with the character picker.
